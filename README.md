# 屁孩特攻隊 — 小孩任務中心

一個給家長與小孩用的家庭任務 PWA：放電運動、體能闖關、晨間/睡前流程、家事輪盤、狀態紀錄，搭配星星獎勵與成就，幫小孩養成好習慣。

> 純前端網頁 App（PWA），可「加入主畫面」當成 App 用。資料存在手機本機，並可選擇性同步到自架的雲端後端。**不做任何醫療診斷，只做家庭生活觀察。**

## 功能

- **五大模組**：今日放電任務、體能闖關、晨間/睡前流程、家事任務輪盤、今日狀態紀錄
- **多小孩**：每個小孩獨立資料、頭像顏色、年齡層
- **星星系統**：完成任務得星、兌換商店、成就徽章、連續天數
- **防刷機制**：每個任務每天只給一次星；闖關每關一輩子只給一次
- **題庫自訂**：放電動作 / 闖關關卡 / 家事 / 獎勵 皆可新增、編輯、刪除
- **天氣建議**：依所在地天氣建議室內/戶外（需後端，否則用免費全球資料）
- **AI 生成**（選用）：用 AI 產生放電動作、關卡、家事、流程步驟、獎勵
- **雲端同步**（選用）：全家用同一組同步碼，逐小孩合併、近即時、不互相覆蓋
- **即時編輯鎖**（選用）：同一個小孩同時只有一台能編輯，避免衝突
- **家長工具**：管理頁 PIN 鎖、星星手動調整、重置進度、資料備份匯出/匯入
- **其他**：語音輸入、深色模式、底部導覽列、離線可用

## 技術架構

純前端、無建置工具：

| 檔案 | 說明 |
|---|---|
| `index.html` | App 外殼 |
| `styles.css` | 樣式（活潑童趣風 + 深色模式） |
| `data.js` | 預設內容（動作/關卡/流程/家事/獎勵） |
| `weather.js` | 天氣模組 |
| `app.js` | 主程式（hash 路由 + localStorage 狀態） |
| `sw.js` | Service Worker（network-first，離線快取） |
| `manifest.json` / `icon-*.png` | PWA 設定與圖示 |
| `server.js` | 本機測試用靜態伺服器 |
| `worker/` | 選用後端（Cloudflare Worker：AI 生成、天氣、雲端同步、即時鎖） |

資料儲存：預設全部存在裝置的 `localStorage`；啟用雲端同步後，會透過後端把各小孩資料合併到雲端（Cloudflare KV）。

## 本機執行

```bash
node server.js
# 開 http://localhost:8731
```

## 部署（前端）

任何靜態空間皆可（GitHub Pages / Netlify / Cloudflare Pages…）。
更新版本時記得把 `sw.js` 的 `CACHE` 版號 +1，使用者重開 App 會自動更新。

## 選用後端（AI / 天氣 / 雲端同步 / 即時鎖）

後端是一支 Cloudflare Worker，部署與設定方式見 [`worker/README.md`](worker/README.md)。

- **所有金鑰（AI、氣象）只存在 Cloudflare Secret，不寫進程式碼、不進 git。**
- 前端只在「管理頁」填入後端網址與同步碼，這些屬於使用者個人設定，存在各自裝置本機，不包含在原始碼中。
- 後端僅接受來自本 App 網域的請求（Origin 驗證）。

不設定後端也能完整使用：天氣改用免費全球資料、AI/同步/即時鎖等功能自動隱藏。

## 隱私

- 小孩資料只有暱稱、年齡層、任務完成狀況，無敏感個資。
- 不啟用雲端同步時，資料完全不離開裝置。
- 啟用同步時，資料存於使用者自己的 Cloudflare 帳號空間，以自訂同步碼存取。
